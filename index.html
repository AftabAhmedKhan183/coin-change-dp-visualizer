▸ PROBLEM STATEMENT

Given an array of coin denominations coins[] and a target amount V, find the total number of distinct ways to make exactly V using unlimited coins of each denomination. Order does not matter — 1+2 and 2+1 count as one way.

▸ DP RECURRENCE
dp[i][j] = ways to make amount j using first i coin types
dp[i][j] = dp[i−1][j] (exclude coin i) + dp[i][j−coins[i]] (include coin i)
Base case: dp[i][0] = 1 for all i  (one way to make 0: use no coins)

▸ PSEUDOCODE
function countWays(coins[], V):
  n = length(coins)

  // Initialize DP table
  dp[0..n][0..V] = 0

  // Base case: 1 way to make amount 0
  for i = 0 to n:
    dp[i][0] = 1

  // Fill table bottom-up
  for i = 1 to n:
    for j = 1 to V:
      dp[i][j] = dp[i-1][j]               // exclude coins[i]
      if coins[i-1] <= j:
        dp[i][j] += dp[i][j - coins[i-1]] // include coins[i]

  return dp[n][V]   // final answer
